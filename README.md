# YUEJIN — Luxury wealth-gaming landing

Production-ready Next.js landing page: user acquisition plus investor narrative, styled as a high-end fintech / wealth ecosystem with global positioning.

## Prerequisites

- Node.js 20+ recommended
- npm (bundled with Node)

## Run locally

From this folder (`yuejin-web`):

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Scripts

- `npm run dev` — development server
- `npm run build` — production build
- `npm run start` — serve production build (after `build`)
- `npm run lint` — ESLint

## Stack

- Next.js (App Router)
- Tailwind CSS v4
- Framer Motion

## Brand

- **Name:** YUEJIN (“leaping gold” — prosperity motif)
- **Mark:** Inline SVG in `components/brand/Logo.tsx`
- **Palette:** Deep red `#8B0000`, gold `#D4AF37`, void black, emerald accent
